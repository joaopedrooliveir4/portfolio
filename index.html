Tecnologias:

HTML

Linguagem de marcação usada para estruturar conteúdo na web, definindo elementos como textos, imagens e links.

CSS

Folhas de estilo em cascata que controlam a apresentação visual de páginas web, permitindo o design e o layout.

JAVASCRIPT

Linguagem de programação que permite adicionar interatividade e dinamismo às páginas web, manipulando o DOM e respondendo a eventos do usuário.

BOOTSTRAP

Framework front-end que facilita o desenvolvimento de sites responsivos e móveis, oferecendo uma coleção de estilos e componentes pré-definidos.

PYTHON

Linguagem de programação versátil e de alto nível, popular em ciência de dados, automação, desenvolvimento web e aprendizado de máquina.

NODE.JS

Ambiente de execução JavaScript do lado do servidor que permite criar aplicações de rede escaláveis, usando eventos assíncronos e um modelo baseado em I/O não bloqueante.
